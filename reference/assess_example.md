# Larger example dataset for further runs

Sufficient for assess_run()

## Usage

``` r
assess_example
```

## Format

### `assess_example`

A list with of three tibbles:

- metadata:

  Example metadata tibble

- asvtable:

  Example ASV count table

- taxa:

  Example ASV taxonomy table
