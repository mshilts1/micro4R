# Filter the ASV table

Filter the ASV table

## Usage

``` r
filtering(asvtable = NULL, minDepth = 1000, minASVCount = 2)
```

## Arguments

- asvtable:

  Input ASV table

- minDepth:

  Minimum number of reads a sample must have to be kept

- minASVCount:

  Minimum count an ASV must have to be kept

## Value

A filtered ASV table
